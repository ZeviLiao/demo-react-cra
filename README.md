emotion:  

https://emotion.sh/docs/install  

```
yarn add @emotion/react
yarn add @emotion/styled
yarn add --dev @emotion/babel-plugin
```

.babelrc  in project root  
```
{
  "plugins": ["@emotion"]
}
```

reset css
```
yarn add normalize.css
```

in index.js file add line of code.
```
@import-normalize;

```
